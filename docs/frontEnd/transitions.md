## Usage
The components can be used outside of the Enso ecosystem.

### bulma/Fade.vue
A reusable fade transition.

Slots:
- `default`, slot for the actual contents

### transitions/FadeDown.vue
A reusable fade down transition.

Slots:
- `default`, slot for the actual contents

### transitions/FadeLeft.vue

Slots:
- `default`, slot for the actual contents
