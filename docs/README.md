### Try it up

``` bash
# clone in a new project folder
git clone https://github.com/laravel-enso/Enso.git ./

# install composer dependencies
composer install

# Configure the .env file
# Generate the key
php artisan key:generate

# run the migrations
php artisan migrate

# install node dependencies
yarn

# compile
yarn dev / yarn hot
```
